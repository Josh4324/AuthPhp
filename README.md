### How to Deploy

1. Clone Repo - git clone https://github.com/Josh4324/AuthPhp.git

2. Install XAMMP

3. Copy the cloned folder into the XAMMP server directory i.e C:\xampp\htdocs\

4. Open the XAMMP control panel and start the apache server
